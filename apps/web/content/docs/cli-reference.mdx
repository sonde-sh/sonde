## Install

```sh
npm install -g @sonde-sh/sonde
```

Or execute directly:

```sh
npx @sonde-sh/sonde --help
```

## Global flags

- `--json`: emit machine-readable JSON payloads.
- `--help`, `-h`: show command usage.
- `--version`, `-v`: print package version.

## Commands

### `sonde generate <cli> [--json]`

Generate `sondage.manifest.json` from `<cli> --help` output.

### `sonde run <cli> [--json]`

Run deterministic checks using the local `sondage.manifest.json`.

### `sonde score <cli> [--json]`

Compute a weighted CLI score (0-100) from manifest + run behavior.

### `sonde serve [--json]`

Run a line-delimited JSON protocol for tool discovery/execution.

## Prompt control

Use explicit answers, non-interactive modes, or skip flags in CI.

## Artifact export

Expose report and manifest paths so tooling can discover them automatically.

## Operational guidance

Sonde can expose generated artifacts through MCP-compatible surfaces so docs,
automation, and external tooling consume a single source of truth.
