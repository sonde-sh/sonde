description: Sonde defines an STM-oriented model for evaluating CLI behavior in repeatable, machine-readable scenarios.

## Key Elements

### State transitions

Model deterministic execution stages for each CLI scenario.

### Observations

Capture stdout/stderr, JSON payloads, and prompt interactions.

### Scoring

Compute comparable totals based on explicit criteria and weights.

## When to use

Use this spec when you need deterministic test assertions across different CLIs,
environments, or CI runs without relying on free-form manual interpretation.
