# Security Policy

## Supported versions

Only the latest release is supported for security updates.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities through GitHub Private Vulnerability Reporting:

- https://github.com/sonde-sh/sonde/security/advisories/new

Include:

- A clear description of the issue
- Reproduction steps or a proof of concept
- Potential impact
- Any suggested remediation

If your repository fork does not have private advisory access, open the
security issue template and we will immediately move the report to a private
channel.

We will acknowledge receipt within 3 business days and provide status updates
as triage progresses.
